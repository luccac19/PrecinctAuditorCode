# PrecinctAuditorCode
This is a repository for sharing the code to run precinct/district auditing program locally on any device using R. [Interface Code](InterfaceCode.R) contains the code for the Shiny app interface that makes the program so easy to upload address and boundary files to. [Output Code](OutputCode.Rmd) contains the code for processing those files into an html output file that can be easily printed and marked as you further investigate addresses. If you do not have a device with R, no problem! Go to the web-hosted app at https://politicalscience.shinyapps.io/PrecinctAuditor/
